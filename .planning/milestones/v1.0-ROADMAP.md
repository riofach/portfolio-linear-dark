## Milestone Summary

**Key Decisions:**
- **Hybrid SPA approach:** Server Components for shell, Client for motion.
- **Content in `data/*.ts`:** Maintainability without CMS complexity.
- **Tailwind v4 CSS-first:** Standardize styling.
- **CSS 3D Transforms:** Used for Hero grid instead of WebGL for performance.
- **Visual-only Contact Form:** Deferred backend integration to later phase.
- **Data-driven Skills Grid:** Organized skills by category for better structure.
- **Period & Tags fields:** Standardized data structure for timeline and cards.
- **Unified Max-Width:** Consistent alignment across all sections.
- **Skills Data in .tsx:** Support JSX/SVG directly in data for maintenance.
- **Experience Data Separation:** Separated content from presentation with icon mapping.
- **Native Scroll:** Removed Lenis for simplicity and robustness.
- **Centralized Motion:** Used `src/lib/motion.ts` for consistent physics.
- **Staggered Hero Entrance:** Created immediate visual hierarchy on load.
- **Static Hero Background:** Kept grid static to avoid distracting from text entrance.
- **Hover Scale:** Used 1.05 for Skills, 1.02 for Projects.
- **Simulated Backend:** Used timeout for contact form to mimic API calls.
- **Staggered Entrance:** Applied `STAGGER` constant to Experience items.
- **Metadata API:** Used Next.js 14+ Metadata API for SEO.
- **Canonical Domain:** Targeted 'rioraditya.click' to consolidate ranking signals.
- **Entity Recognition:** Used full name in Title/JSON-LD for personal branding.
- **MotionLink Pattern:** Wrapped `next/link` with `motion.create()` for interactive cards.
- **Smart Parent / Dumb Child:** Separated layout/entrance (Parent) from visuals/hover (Child).
- **Remote Patterns:** Whitelisted `lh3.googleusercontent.com` for project images.

**Issues Resolved:**
- **Fixed Font Rendering:** Addressed in Phase 1 Gap Closure.
- **Structure Cleanup:** Refactored layout in Phase 2.
- **Technical SEO:** Added in Phase 3 Gap Closure.
- **Image Config:** Fixed in Phase 4 Gap Closure.

**Technical Debt Incurred:**
- **Experience Link:** Missing from main navigation links (Phase 1 tech debt).
- **Contact Form Backend:** Mocked with `setTimeout` (Phase 3 tech debt, to be addressed in v2).
