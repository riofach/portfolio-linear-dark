# Sanity Setup & Configuration

Initialize the Sanity client and Studio within the Next.js application.

## Tasks

### 1. Install Dependencies
Install `next-sanity`, `@sanity/image-url`, and `sanity`.
-   Run `npm install next-sanity @sanity/image-url sanity`

### 2. Configure Sanity
Create the configuration files.
-   Create `sanity.config.ts` in the root.
-   Create `sanity.cli.ts` in the root.
-   Add `NEXT_PUBLIC_SANITY_PROJECT_ID` and `NEXT_PUBLIC_SANITY_DATASET` to `.env.local`.

### 3. Create Studio Route
Set up the route to render the Sanity Studio.
-   Create `app/studio/[[...index]]/page.tsx`.
-   Implement `NextStudio` component.
