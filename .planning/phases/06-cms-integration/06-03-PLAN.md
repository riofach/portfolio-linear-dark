# Supplemental Schemas

Define blog posts, global settings, and internationalization helpers.

## Tasks

### 1. Create Post Schema
Define the schema for blog posts.
-   Create `sanity/schemas/documents/post.ts`.
-   Fields: title, slug, publishedAt, mainImage, body (block content), author.

### 2. Create Settings Singleton
Define global site settings.
-   Create `sanity/schemas/singletons/settings.ts`.
-   Fields: siteTitle, seoDescription, socialLinks, heroText (localized).

### 3. Create Localization Helpers
Implement schema objects for localized fields (Indonesian/English).
-   Create `sanity/schemas/objects/localizedString.ts`.
-   Create `sanity/schemas/objects/localizedText.ts`.
-   Register these in `sanity/schema.ts`.
