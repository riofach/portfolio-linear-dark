# UI Integration

Connect the frontend components to the Sanity data layer.

## Tasks

### 1. Integrate Homepage
Replace static data on the Landing Page.
-   Update `app/[locale]/page.tsx`.
-   Fetch and display `heroText` and `featuredProjects`.

### 2. Integrate Projects Page
Update the Projects listing page.
-   Update `app/[locale]/projects/page.tsx`.
-   Fetch and render the full list of `projects`.

### 3. Integrate Blog Page
Update the Blog listing page.
-   Update `app/[locale]/blog/page.tsx` (or `posts`).
-   Fetch and render the list of `posts`.
